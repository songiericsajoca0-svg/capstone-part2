💰 FARE AUTOMATION FORMULA

📐 DISTANCE CALCULATION (Haversine Formula): distance = 6371 × arccos[ sin(φ₁) × sin(φ₂) + cos(φ₁) × cos(φ₂) × cos(Δλ) ] 💵 FARE BREAKDOWN: IF total_pax ≤ 2: base_fare = ₱30 ELSE: base_fare = total_pax × ₱15 pickup_fee = CEIL(distance_km) × ₱10 total_fare = base_fare + pickup_fee required_tricycles = CEIL(total_pax / 4) 📊 EXAMPLE (3 passengers, 2.3 km): base_fare = 3 × 15 = ₱45 | rounded_distance = 3 km | pickup_fee = 3 × 10 = ₱30 TOTAL = ₱75

🛡️ HIGHWAY RESTRICTION LOGIC

INPUT: pickup_coords, dropoff_coords STEP 1: Check EDSA geofence (14.54-14.70 lat, 121.00-121.07 lng) STEP 2: Call OSRM routing API to get road names STEP 3: Check each road segment against restricted list: [EDSA, C5, NLEX, SLEX, Skyway, Commonwealth, Roxas Blvd, etc.] STEP 4: If ANY restricted road found → BLOCK booking Else → ALLOW booking
